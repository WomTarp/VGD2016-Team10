Special conditions for playing the game:

1) Take the Le Musiciones file directory (the whole directory)
   and place it in:
   C:\Users\YOUR_USERNAME\AppData\LocalLow

2) Play the game in 400x300 windowed mode. There is a specific
   GUI aspect in the middle of the game that doesn't rescale well.
   Also, the main character's speed was optimized for this resolution.
